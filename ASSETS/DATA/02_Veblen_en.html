Definition

  Veblen Function refers to an ordinal function proposed by Oswald Veblen in 1908[1]. The definition of the function \( \varphi_\alpha(\beta) \) is as follows:

  01. \( \varphi_0(\beta)=\omega^\beta \)

  02. For a non-zero ordinal \( \alpha \), \( \varphi_{\alpha}(\beta) \) returns the \( (1+\beta) \)-th common fixed point of all of the functions \( \gamma\mapsto\varphi_{\alpha'}(\gamma) \) for \( \alpha'<\alpha \).

  A fixed point of a function \( f \) is an input of the function \( x \) that \( x=f(x) \).


  The normal form of this system is called Veblen Normal Form and is defined as follows:

  \( \alpha=\varphi_{\alpha_0}(\beta_0)+\varphi_{\alpha_1}(\beta_1)+\cdots+\varphi_{\alpha_n}(\beta_n) \)

  Where \( n \) is a natural number, \( \varphi_{\alpha_0}(\beta_0)\ge\varphi_{\alpha_1}(\beta_1)\ge\cdots\ge\varphi_{\alpha_n}(\beta_n) \) and \( \varphi_{\alpha_m}(\beta_m)<\beta_m \) for all natural numbers \( m\le n \).

  Every non-zero ordinal can be uniquely written as Veblen Normal Form.


  In this website, we are working with ordinals less than the Feferman-Sch√ºtte ordinal \( \Gamma_0 \) because \( \Gamma_0 \) is the least ordinal that \( \alpha=\varphi_{\alpha}(0) \) and every ordinal less than \( \Gamma_0 \) can be constructed only using zero, addition and the function.

  The fundamental sequence system used in this website is defined as follows:

  01. \( 0[n]:=0 \)

  02. \( (\varphi_{\alpha_0}(\beta_0)+\varphi_{\alpha_1}(\beta_1)+\cdots+\varphi_{\alpha_m}(\beta_m)+\varphi_{\alpha_{m+1}}(\beta_{m+1}))[n]:=\varphi_{\alpha_0}(\beta_0)+\varphi_{\alpha_1}(\beta_1)+\cdots+\varphi_{\alpha_m}(\beta_m)+\varphi_{\alpha_{m+1}}(\beta_{m+1})[n] \)

  03. \( \varphi_0(0)[n]=1[n]:=0 \)

  04. \( \varphi_0(\beta+1)[n]:=\varphi_0(\beta)*n \)

  05. \( \varphi_{\alpha+1}(0)[0]:=0 \) and \( \varphi_{\alpha+1}(0)[n+1]:=\varphi_\alpha(\varphi_{\alpha+1}(0)[n]) \)

  06. \( \varphi_\alpha(0)[n]:=\varphi_{\alpha[n]}(0) \) (\( \alpha \) is a limit ordinal)

  07. \( \varphi_1(\beta+1)[0]:=0 \), \( \varphi_1(\beta+1)[1]:=\varphi_1(\beta) \), \( \varphi_1(\beta+1)[2]:=\varphi_0(\varphi_1(\beta)*2) \) and \( \varphi_1(\beta+1)[n+3]:=\varphi_0(\varphi_1(\beta+1)[n+2]) \)

  08. \( \varphi_{\alpha+1}(\beta+1)[0]:=0 \), \( \varphi_{\alpha+1}(\beta+1)[1]:=\varphi_{\alpha+1}(\beta) \), \( \varphi_{\alpha+1}(\beta+1)[2]:=\varphi_\alpha(\varphi_{\alpha+1}(\beta)+1) \) and \( \varphi_{\alpha+1}(\beta+1)[n+3]:=\varphi_\alpha(\varphi_{\alpha+1}(\beta+1)[n+2]) \) (\( \alpha\neq0 \))

  09. \( \varphi_\alpha(\beta+1)[n]:=\varphi_{\alpha[n]}(\varphi_\alpha(\beta)+1) \) (\( \alpha \) is a limit ordinal)

  10. \( \varphi_\alpha(\beta)[n]:=\varphi_\alpha(\beta[n]) \) (\( \beta \) is a limit ordinal)

  11. \( \Gamma_0[0]:=0 \) and \( \Gamma_0[n+1]:=\varphi_{\Gamma_0[n]}(0) \)


  Note that there's also another fundamental sequence system which is more well-known called the Veblen hierarchy defined as follows, which slightly differs from this one. The reason why the fundamental sequence system above is used is to mimic the fundamental sequence system of Buchholz's \( \psi \) and its extension:

  01. \( 0[n]:=0 \)

  02. \( (\varphi_{\alpha_0}(\beta_0)+\varphi_{\alpha_1}(\beta_1)+\cdots+\varphi_{\alpha_m}(\beta_m)+\varphi_{\alpha_{m+1}}(\beta_{m+1}))[n]:=\varphi_{\alpha_0}(\beta_0)+\varphi_{\alpha_1}(\beta_1)+\cdots+\varphi_{\alpha_m}(\beta_m)+\varphi_{\alpha_{m+1}}(\beta_{m+1})[n] \)

  03. \( \varphi_0(0)[n]=1[n]:=0 \)

  04. \( \varphi_0(\beta+1)[n]:=\varphi_0(\beta)*n \)

  05. \( \varphi_{\alpha+1}(0)[0]:=0 \) and \( \varphi_{\alpha+1}(0)[n+1]:=\varphi_\alpha(\varphi_{\alpha+1}(0)[n]) \)

  06. \( \varphi_\alpha(0)[n]:=\varphi_{\alpha[n]}(0) \) (\( \alpha \) is a limit ordinal)

  07. \( \varphi_{\alpha+1}(\beta+1)[0]:=\varphi_{\alpha+1}(\beta)+1 \) and \( \varphi_{\alpha+1}(\beta+1)[n+1]:=\varphi_\alpha(\varphi_{\alpha+1}(\beta+1)[n]) \)

  08. \( \varphi_\alpha(\beta+1)[n]:=\varphi_{\alpha[n]}(\varphi_\alpha(\beta)+1) \) (\( \alpha \) is a limit ordinal)

  09. \( \varphi_\alpha(\beta)[n]:=\varphi_\alpha(\beta[n]) \) (\( \beta \) is a limit ordinal)

  10. \( \Gamma_0[0]:=0 \) and \( \Gamma_0[n+1]:=\varphi_{\Gamma_0[n]}(0) \)


  Sometimes, we rewrite the function as follows:

  \( \omega^\beta:=\varphi_0(\beta) \)

  \( \varepsilon_\beta:=\varphi_1(\beta) \)

  \( \zeta_\beta:=\varphi_2(\beta) \)

  \( \eta_\beta:=\varphi_3(\beta) \)

Ordinal Notation

Notation

  We define \( S \) to be the set that contains all the strings composed of the following symbols:

  Constant "\( \overline{0} \)",

  Function symbol "\( \varphi \)",

  And other symbols "\( + \)", "\( ( \)", "\( ) \)" and "\( , \)".


  We define the subsets of \( S \), \( T_m \) and \( T_s \) as follows:

  01. \( \overline{0}\in T_m \)

  02. For all \( a\in T_s \) and \( b\in T_m\setminus\{\overline{0}\} \), \( a+b\in T_m \)

  03. For all \( a,b\in T_m \), \( \varphi(a,b)\in T_m \) and \( \varphi(a,b)\in T_s \)


  Throughout this article, we use the following abbreviation:

  01. \( \varphi_a(b):=\varphi(a,b) \)

  02. \( \overline{1}:=\varphi_\overline{0}(\overline{0}) \)

  03. \( \overline{\omega}:=\varphi_\overline{0}(\overline{1}) \)

Ordering

  We define the binary relation \( < \) as follows:

  \( (a\in T_m,b\in T_m)\mapsto a<b\in\{\mathbb{T},\mathbb{F}\} \)

  01. If \( b=\overline{0} \), return \( \mathbb{F} \)

  02. If \( a=\overline{0}\land b\neq\overline{0} \), return \( \mathbb{T} \)

  03. If \( a=a_0+a_1\land b=b_0+b_1\land a_0,b_0\in T_s\land a_1,b_1\in T_m\setminus\{\overline{0}\} \), then:

  03.01. If \( a_0<b_0 \), return \( \mathbb{T} \)

  03.02. If \( \neg a_0<b_0 \), return \( \neg b_0<a_0\land a_1<b_1 \)

  04. If \( a=a_0+a_1\land b,a_0\in T_s\land a_1\in T_m\setminus\{\overline{0}\} \), return \( a_0<b \)

  05. If \( b=b_0+b_1\land a,b_0\in T_s\land b_1\in T_m\setminus\{\overline{0}\} \), return \( \neg b_0<a \)

  06. If \( a=\varphi_{a_0}(a_1)\land b=\varphi_{b_0}(b_1)\land a_0,a_1,b_0,b_1\in T_m \), then:

  06.01. If \( a_0<b_0 \), return \( a_1<b \)

  06.02. If \( b_0<a_0 \), return \( a<b_1 \)

  06.03. If \( a_0=_*b_0 \), return \( a_1<b_1 \)


  We define all the other comparison symbols:

  \( (a\in T_m,b\in T_m)\mapsto a>b\in\{\mathbb{T},\mathbb{F}\} \)

  Return \( b<a \)


  \( (a\in T_m,b\in T_m)\mapsto a\le b\in\{\mathbb{T},\mathbb{F}\} \)

  Return \( \neg b<a \)


  \( (a\in T_m,b\in T_m)\mapsto a\ge b\in\{\mathbb{T},\mathbb{F}\} \)

  Return \( \neg a<b \)


  \( (a\in T_m,b\in T_m)\mapsto a=_*b\in\{\mathbb{T},\mathbb{F}\} \)

  Return \( \neg a<b\land\neg b<a \)


  \( (a\in T_m,b\in T_m)\mapsto a\neq_*b\in\{\mathbb{T},\mathbb{F}\} \)

  Return \( a<b\vee b<a \)


  Note that \( = \) and \( \neq \) represents symbolic equivalence while \( =_* \) and \( \neq_* \) represents numerical equivalence.

Normal Form

  We define the function \( \in T \) to determine whether an ordinal is in normal form:

  \( (a\in T_m)\mapsto (a\in T)\in\{\mathbb{T},\mathbb{F}\} \)

  01. If \( a=\overline{0} \), return \( \mathbb{T} \)

  02. If \( a=a_0+a_1\land a_0\in T_s\land a_1\in T_m\setminus\{\overline{0}\} \), then:

  02.01. If \( \neg a_0\in T\vee\neg a_1\in T \), return \( \mathbb{F} \)

  02.02. If \( a_0\in T\land a_1\in T \), then:

  02.02.01. If \( a_1\in T_s \), return \( \neg a_0<a_1 \)

  02.02.02. If \( a_1=a_2+a_3\land a_2\in T_s\land a_3\in T_m\setminus\{\overline{0}\} \), return \( \neg a_0<a_2 \)

  03. If \( a=\varphi_{a_0}(a_1)\land a_0,a_1\in T_m \), return \( a_0\in T\land a_1\in T\land a_0<a\land a_1<a \)


  We define the limit of the system \( A=\Gamma_0 \).

  \( \Gamma_0 \) is also in normal form.

  For all ordinals in this system \( \alpha\in T \), \( \alpha<\Gamma_0 \)

  Ordinals in this system are always countable.

  There is no countable limit ordinal exclusive to the command "fgh" or "sgh".

Operations

  We define the normalization function \( Norm \) to normalize an ordinal to its normal form:

  \( (a\in T_m)\mapsto Norm(a)\in T \)

  01. If \( a\in T \), return \( a \)

  02. If \( \neg a\in T \), then:

  02.01. If \( a=a_0+a_1\land a_0\in T_s\land a_1\in T_m\setminus\{\overline{0}\} \), return \( Norm(a_0)+Norm(a_1) \)

  02.02. If \( a=\varphi_{a_0}(a_1)\land a_0,a_1\in T_m \), then:

  02.02.01. If \( a=a_1 \), return \( a_1 \)

  02.02.02. If \( a=a_0 \), return \( a_0 \)

  02.02.03. If \( a\neq a_0\land a\neq a_1 \), return \( a \)


  We define addition \( Add \) as follows:

  \( (a\in T,b\in T)\mapsto Add(a,b)\in T \)

  01. If \( a=\overline{0} \), return \( b \)

  02. If \( a\neq\overline{0}\land b=\overline{0} \), return \( a \)

  03. If \( a\neq\overline{0}\land b\neq\overline{0} \), return \( a+b \)


  We define succession \( Suc \) as follows:

  \( (a\in T)\mapsto Suc(a)\in T\setminus\{\overline{0}\} \)

  Return \( Add(a,\overline{1}) \)

Cofinality

  We define the cofinality function \( Cof \) as follows:

  \( (a\in T)\mapsto Cof(a)\in T \)

  01. If \( a=\overline{0} \), return \( a \)

  02. If \( a=a_0+a_1\land a_0\in T_s\land a_1\in T_m\setminus\{\overline{0}\} \), return \( Cof(a_1) \)

  03. If \( a=\varphi_{a_0}(a_1)\land a_0,a_1\in T_m \), then:

  03.01. If \( \neg Cof(a_1)<\overline{\omega} \), return \( Cof(a_1) \)

  03.02. If \( \neg Cof(a_0)<\overline{\omega}\land Cof(a_1)<\overline{\omega} \), return \( Cof(a_0) \)

  03.03. If \( Cof(a_0)<\overline{\omega}\land Cof(a_1)<\overline{\omega}\land(a_0\neq\overline{0}\vee a_1\neq\overline{0}) \), return \( \overline{\omega} \)

  03.04. If \( a_0=\overline{0}\land a_1=\overline{0} \), return \( \overline{1} \)


  We also define that \( Cof(\Gamma_0)=\overline{\omega} \).

Fundamental Sequence

  We define the fundamental sequence \( [] \) as follows:

  \( (a\in T,n\in T)\mapsto a[n]\in T \)

  01. If \( a=\overline{0} \), return \( \overline{0} \)

  02. If \( a=a_0+a_1\land a_0\in T_s\land a_1\in T_m\setminus\{\overline{0}\} \), return \( Add(a_0,a_1[n]) \)

  03. If \( a=\varphi_{a_0}(a_1)\land a_0,a_1\in T_m \), then:

  03.01. If \( a_1=\overline{0} \), then:

  03.01.01. If \( a_0=\overline{0} \), return \( \overline{0} \)

  03.01.02. If \( Cof(a_0)=\overline{1} \), then:

  03.01.02.01. If \( n=\overline{0} \), return \( \overline{0} \)

  03.01.02.02. If \( n\neq\overline{0} \), return \( \varphi_{a_0[\overline{0}]}(a[n[\overline{0}]]) \)

  03.01.03. If \( Cof(a_0)\ge\overline{\omega} \), return \( Norm(\varphi_{a_0[n]}(\overline{0})) \)

  03.02. If \( Cof(a_1)=\overline{1} \), then:

  03.02.01. If \( a_0=\overline{0} \), then:

  03.02.01.01. If \( n=\overline{0} \), return \( \overline{0} \)

  03.02.01.02. If \( n\neq\overline{0} \), return \( Add(a[n[\overline{0}]], \varphi_\overline{0}(a_1[\overline{0}])) \)

  03.02.02. If \( Cof(a_0)=\overline{1} \), then:

  03.02.02.01. If \( n=\overline{0} \), return \( \overline{0} \)

  03.02.02.02. If \( n\neq\overline{0}\land n[\overline{0}]=\overline{0} \), return \( Norm(\varphi_{a_0}(a_1[\overline{0}])) \)

  03.02.02.03. If \( n[\overline{0}]\neq\overline{0}\land n[\overline{0}][\overline{0}]=\overline{0} \), then:

  03.02.02.03.01. If \( a_0=\overline{1} \), return \( \varphi_\overline{0}(a[n[\overline{0}]]+a[n[\overline{0}]]) \)

  03.02.02.03.02. If \( a_0\neq\overline{1} \), return \( Norm(\varphi_{a_0[\overline{0}]}(Suc(a[n[\overline{0}]]))) \)

  03.02.02.04. If \( n[\overline{0}][\overline{0}]\neq\overline{0} \), return \( \varphi_{a_0[\overline{0}]}(a[n[\overline{0}]]) \)

  03.02.03. If \( Cof(a_0)\ge\overline{\omega} \), return \( \varphi_{a_0[n]}(Suc(Norm(\varphi_{a_0}(a_1[\overline{0}])))) \)

  03.03. If \( Cof(a_1)\ge\overline{\omega} \), return \( Norm(\varphi_{a_0}(a_1[n])) \)


  We also define the fundamental sequence of \( \Gamma_0 \) as follows:

  \( (n\in T)\mapsto\Gamma_0[n]\in T \)

  01. If \( n=\overline{0} \), return \( \overline{0} \)

  02. If \( n\neq\overline{0} \), return \( \varphi_{\Gamma_0[n[\overline{0}]]}(0) \)
