Definition

  Rathjen's Capital \( \Psi \) refers to another ordinal collapsing function proposed by Michael Rathjen in 1994[1]. The whole ordinal system consists of 4 functions: \( \varphi_\alpha(\beta) \), \( \Omega_\alpha \), \( \Xi(\alpha) \) and \( \Psi_\alpha^\beta(\gamma) \) and described and defined as follows.

The function \( \varphi_\alpha(\beta) \)

  \( \varphi_\alpha(\beta) \) denotes Veblen Function. It's defined as follows as shown in the Help menu in the "Veblen Function" interface:

  01. \( \varphi_0(\beta)=\omega^\beta \)

  02. For a non-zero ordinal \( \alpha \), \( \varphi_{\alpha}(\beta) \) returns the \( (1+\beta) \)-th common fixed point of all of the functions \( \gamma\mapsto\varphi_{\alpha'}(\gamma) \) for \( \alpha'<\alpha \).

The function \( \Omega_\alpha \)

  \( \Omega_\alpha \) denotes the \( (1+\alpha) \)-th infinite cardinal. Note that \( \Omega_0 \) is defined as \( 0 \) in the paper.

The function \( \Xi(\alpha) \)

  \( \Xi(\alpha) \) denotes the least element in the Mahlo Cardinal hierarchy \( M^\alpha \), which can be considered the "least Mahlo function". It's defined as follows: (\( \alpha<\Gamma_{K+1} \))

  \( \Xi(\alpha)=min(M^\alpha\cup\{K\}) \)


  The class \( M^\alpha \) is defined as follows:

  \( M^0=Lim\cap K \)

  \( \begin{align} M^\alpha=\{\kappa| & \kappa\in Reg\cap K \\ & C(\alpha,\kappa)\cap K=\kappa \\ & \alpha\in C(\alpha,\kappa) \\ & \forall\alpha'\in C(\alpha,\kappa)\cap\alpha(M^{\alpha'}\in Sta_\kappa) \}\ (\alpha>0) \end{align} \)

  Where \( Lim \) denotes the class of limit ordinals, \( Reg \) denotes the class of regular cardinals and \( Sta_\kappa \) denotes the class of sets stationary in \( \kappa \). You can check the definition of stationary sets in the Help menu in the "Rathjen's Little \( \psi \)" interface.


  The set \( C(\alpha,\beta) \) is defined as follows:

  \( \begin{align} C_0(\alpha,\beta) := & \beta\cup\{0,K\} \\ C_{n+1}(\alpha,\beta) := & \{\gamma|\gamma=\gamma_0+\gamma_1\land\gamma_0,\gamma_1\in C_n(\alpha,\beta)\} \\ \cup & \{\gamma|\gamma=\varphi_{\alpha'}(\beta')\land\alpha',\beta'\in C_n(\alpha,\beta)\} \\ \cup & \{\gamma|\gamma=\Omega_{\alpha'}\land\alpha'\in C_n(\alpha,\beta)\cap K\} \\ \cup & \{\gamma|\gamma=\Xi(\alpha')\land\alpha'\in C_n(\alpha,\beta)\cap\alpha\} \\ \cup & \{\gamma|\gamma=\Psi_{\alpha'}^{\beta'}(\gamma')\land\alpha'\in Reg\land\beta'\le\gamma'<\alpha\land\alpha',\beta',\gamma'\in C_n(\alpha,\beta) \} \\ C(\alpha,\beta) := & \bigcup_{n<\omega}{C_n(\alpha,\beta)}=C_0(\alpha,\beta)\cup C_1(\alpha,\beta)\cup C_2(\alpha,\beta)\cup\cdots\cup C_m(\alpha,\beta)\cup\cdots \end{align} \)


  From the definition above, we can see that it introduces an extension of Mahlo Cardinals:

  \( M^0 \) denotes the class of limit ordinals (less than \( K \)).

  \( M^1 \) denotes the class of Inaccessible Cardinals due to the presence of the function \( \Omega_\alpha \). The reason why this function is included is probably because Inaccessible Cardinals are equivalent to \( \Pi_0^1- \)indescribable cardinals[2].

  \( M^2 \) denotes the class of Mahlo Cardinals. Not only does this contain the least Mahlo Cardinal \( M \), but it also contains larger Mahlo Cardinals like \( M_2 \), \( M_3 \) and so on.

  \( M^3 \) denotes the class of "Hyper-Mahlo Cardinals". It contains Mahlo Cardinals \( \kappa \) that the set of all Mahlo Cardinals less than \( \kappa \) is a stationary set in \( \kappa \).

  \( M^4 \) denotes the class of "Hyper-Hyper-Mahlo Cardinals". It contains Hyper-Mahlo Cardinals \( \kappa \) that the set of all Hyper-Mahlo Cardinals less than \( \kappa \) is a stationary set in \( \kappa \) and so on.


  As a consequence, \( \Xi(1)=I \) and \( \Xi(2)=M \). Sometimes we denote \( \Xi(3) \) as \( N \).

The function \( \Psi_\alpha^\beta(\gamma) \)

  \( \Psi_\alpha^\beta(\gamma) \) denotes the collapsing function, which is the core of this ordinal system. It's defined as follows: (\( \alpha\in Reg \) and \( \beta\le\gamma \) and \( \gamma<\Gamma_{K+1} \))

  \( \begin{align} \Psi_\alpha^\beta(\gamma)=min(\{\delta| & \delta\in M^\beta\cap\alpha \\ & C(\gamma,\delta)\cap\alpha=\delta \\ & \alpha,\gamma\in C(\gamma,\delta) \}\cup\{\alpha\}) \end{align} \)


  As you can see, this OCF takes 3 ordinals as input instead of 2 like most of other OCFs. If \( \beta>0 \), then \( \Psi_\alpha^\beta(\gamma) \) always return a regular cardinal. Moreover, since both \( \Psi_\alpha^\beta(\gamma) \) and \( \Xi(\beta) \) returns an element of \( M^\beta \), it can represent larger Inaccessible Cardinals, Mahlo Cardinals, "Hyper-Mahlo Cardinals" and so on. As a consequence, \( \Psi_{\Xi(2)}^1(3)=I_2 \), \( \Psi_{\Xi(2)}^1(4)=I_3 \), \( \Psi_{\Xi(3)}^2(4)=M_2 \), \( \Psi_{\Xi(3)}^2(5)=M_3 \), \( \Psi_{\Xi(4)}^3(5)=N_2 \), \( \Psi_{\Xi(4)}^3(6)=N_3 \) and so on.

  Also, it seems it's possible that \( \Psi_\alpha^\beta(\gamma)=\alpha \). In fact, this is true, and same goes for the function \( \Xi(\alpha) \). When this happens, we say that the function "fails to collapse". As a consequence, most of the time, there's a least \( \gamma \) so that \( \Psi_\alpha^\beta(\gamma) \) doesn't fail to collapse.

  The most challenging part of this OCF is that \( \Psi_\alpha^\beta(\gamma) \) doesn't monotonically increase in \( \gamma \), meaning that it's easy to overlook some ordinals when analyzing this OCF.

  For example, we might expect this at first glance: (Where \( supA \) denotes the least ordinal greater or equal to all of the elements in \( A \))

  \( \Psi_\Omega^0(\Omega+1)=sup\{\Psi_\Omega^0(\Omega),\varphi_{\Psi_\Omega^0(\Omega)}(1),\varphi_{\varphi_{\Psi_\Omega^0(\Omega)}(1)}(0),\varphi_{\varphi_{\varphi_{\Psi_\Omega^0(\Omega)}(1)}(0)}(0),\cdots\} \)

  But in fact:

  \( \Psi_\Omega^0(\Omega+1)=sup\{\Psi_\Omega^0(\Omega),\Psi_\Omega^0(\Psi_\Omega^0(\Omega)),\Psi_\Omega^0(\Psi_\Omega^0(\Psi_\Omega^0(\Omega))),\Psi_\Omega^0(\Psi_\Omega^0(\Psi_\Omega^0(\Psi_\Omega^0(\Omega)))),\cdots\} \)

  As a result, it becomes extremely difficult to determine whether an infinite sequence approaches to ordinals represented in this OCF in general. With all the complexities above, Rathjen's Capital \( \Psi \) becomes one of the hardest OCFs to understand.

The Weakly Compact Cardinal \( K \)

  The Weakly Compact Cardinal \( K \) is the key to this OCF. There are multiple definitions of this cardinal. For example, Weakly Compact Cardinals are equivalent to \( \Pi_1^1- \)indescribable cardinals. However, almost all of the definitions require high-level set theory concepts. Since it requires a long article to explain this cardinal clearly, this article doesn't aim to explain this cardinal. One thing is certain, the immense size of this cardinal allows itself to be used to construct high-level Mahlo Cardinals. By the way, ZFC can't determine the existence of Weakly Compact Cardinals.


  The ordinal \( \Psi_\Omega^0(\varepsilon_{K+1}) \) is called the Large Rathjen ordinal (LRO). It's the upper bound of the PTO of \( KP+\Pi_3-reflection \).

Ordering

  \( (a\in T_m,b\in T_m)\mapsto a<b\in\{\mathbb{T},\mathbb{F}\} \)

  02. If \( a=\overline{0}\land b\neq\overline{0} \), return \( \mathbb{T} \)

  03.01. If \( a_0<b_0 \), return \( \mathbb{T} \)

  03.02. If \( \neg a_0<b_0 \), return \( \neg b_0<a_0\land a_1<b_1 \)

  04. If \( a=a_0+a_1\land b,a_0\in T_s\land a_1\in T_m\setminus\{\overline{0}\} \), return \( a_0<b \)

  05. If \( b=b_0+b_1\land a,b_0\in T_s\land b_1\in T_m\setminus\{\overline{0}\} \), return \( \neg b_0<a \)

  06. If \( a=K\land b=K \), return \( \mathbb{F} \)

  07. If \( a=\varphi_{a_0}(a_1)\land b=\varphi_{b_0}(b_1)\land a_0,a_1,b_0,b_1\in T_m \), then:

  07.01. If \( a_0<b_0 \), return \( a_1<b \)

  07.02. If \( b_0<a_0 \), return \( a<b_1 \)

  07.03. If \( a_0=_*b_0 \), return \( a_1<b_1 \)

  08. If \( a=\Omega_{a_0}\land b=\Omega_{b_0}\land a_0,b_0\in T_m \), return \( a_0<b_0 \)

  09. If \( a=\Xi(a_0)\land b=\Xi(b_0)\land a_0,b_0\in T_m \), then:

  09.01. If \( a_0<b_0 \), return \( a_0\in C(b_0,b) \)

  09.02. If \( b_0<a_0 \), return \( \neg b_0\in C(a_0,a) \)

  09.03. If \( a_0=_*b_0 \), return \( \mathbb{F} \)

  10. If \( a=\Psi_{a_0}^{a_1}(a_2)\land b=\Psi_{b_0}^{b_1}(b_2)\land a_0,a_1,a_2,b_0,b_1,b_2\in T_m \), then:

  10.01. If \( b<a_0 \), then:

  10.01.01. If \( a_2<b_2\land a<b_0\land a_0\in C(b_2,b)\land a_1\in C(b_2,b)\land a_2\in C(b_2,b) \), return \( \mathbb{T} \)

  10.01.02. Else if \( \neg a_2<b_2\land\neg(b_0\in C(a_2,a)\land b_1\in C(a_2,a)\land b_2\in C(a_2,a)) \), return \( \mathbb{T} \)

  10.01.03. Else if \( a_0=_*b_0\land a_2=_*b_2\land a_1<b_1\land a_1\in C(b_1,b) \), return \( \mathbb{T} \)

  10.01.04. Else if \( b_1<a_1\land b_1\in C(a_1,a) \), return \( \mathbb{T} \)

  10.01.05. Else, return \( \mathbb{F} \)

  10.02. Else if \( a_0\neg_*b_0\land a_1\neg_*b_1\land a_2\neg_*b_2 \), return \( \neg b<a \)

  10.03. Else, return \( \mathbb{F} \)

  11. If \( a=K\land b\neq K \), then:

  11.01. If \( b=\varphi_{b_0}(b_1)\land b_0,b_1\in T_m \), return \( (a<b_0\vee b_1\neq\overline{0})\land(\neg b_0<a\vee a<b_1) \)

  11.02. If \( b=\Omega_{b_0}\land b_0\in T_m \), return \( a<b_0 \)

  11.03. If \( b=\Xi(b_0)\land b_0\in T_m \), return \( \mathbb{F} \)

  11.04. If \( b=\Psi_{b_0}^{b_1}(b_2)\land b_0,b_1,b_2\in T_m \), return \( \mathbb{F} \)

  12. If \( a\neq K\land b=K \), then:

  12.01. If \( a=\varphi_{a_0}(a_1)\land a_0,a_1\in T_m \), return \( a_0<b\land a_1<b \)

  12.02. If \( a=\Omega_{a_0}\land a_0\in T_m \), return \( a_0<b \)

  12.03. If \( a=\Xi(a_0)\land a_0\in T_m \), return \( \mathbb{T} \)

  12.04. If \( a=\Psi_{a_0}^{a_1}(a_2)\land a_0,a_1,a_2\in T_m \), return \( \mathbb{T} \)

  13. If \( a=\varphi_{a_0}(a_1)\land(b=\Omega_{b_0}\vee b=\Xi(b_0)\vee b=\Psi_{b_0}^{b_1}(b_2))\land a_0,a_1,b_0,b_1,b_2\in T_m \), then:

  13.01. Return \( a_0<b\land a_1<b \)

  14. If \( (a=\Omega_{a_0}\vee a=\Xi(a_0)\vee a=\Psi_{a_0}^{a_1}(a_2))\land b=\varphi_{b_0}(b_1)\land a_0,a_1,a_2,b_0,b_1\in T_m \), then:

  14.01. Return \( (a<b_0\vee b_1\neq\overline{0})\land(\neg b_0<a\vee a<b_1) \)

  15. If \( a=\Omega_{a_0}\land b=\Xi(b_0)\land a_0,b_0\in T_m \), return \( a_0<b \)

  16. If \( a=\Xi(a_0)\land b=\Omega_{b_0}\land a_0,b_0\in T_m \), return \( a<b_0 \)

  17. If \( a=\Omega_{a_0}\land b=\Psi_{b_0}^{b_1}(b_2)\land a_0,b_0,b_1,b_2\in T_m \), then:

  17.01. If \( b_0=\Omega_{b_3}\land b_3\in T_m \), return \( a<b_0 \)

  17.02. Else, return \( a_0<b \)

  18. If \( a=\Psi_{a_0}^{a_1}(a_2)\land b=\Omega_{b_0}\land a_0,a_1,a_2,b_0\in T_m \), then:

  18.01. If \( a_0=\Omega_{a_3}\land a_3\in T_m \), return \( \neg b<a_0 \)

  18.02. Else, return \( a<b_0 \)

  19. If \( a=\Xi(a_0)\land b=\Psi_{b_0}^{b_1}(b_2)\land a_0,b_0,b_1,b_2\in T_m \), return \( a<b_0\land(\neg a_0<b_2\vee a_0\in C(b_2,b)) \)

  20. If \( a=\Psi_{a_0}^{a_1}(a_2)\land b=\Xi(b_0)\land a_0,a_1,a_2,b_0\in T_m \), return \( \neg b<a_0\vee(b_0<a_2\land\neg b_2\in C(a_2,a)) \)


  We define all the other comparison symbols:

  \( (a\in T_m,b\in T_m)\mapsto a>b\in\{\mathbb{T},\mathbb{F}\} \)

  Return \( b<a \)


  \( (a\in T_m,b\in T_m)\mapsto a\le b\in\{\mathbb{T},\mathbb{F}\} \)

  Return \( \neg b<a \)


  \( (a\in T_m,b\in T_m)\mapsto a\ge b\in\{\mathbb{T},\mathbb{F}\} \)

Normal Form

  We define the max function \( Max \) as follows:

  \( (a\in T_m,b\in T_m)\mapsto Max(a,b)\in T_m \)

  01. If \( a<b \), return \( b \)

  02. If \( \neg a<b \), return \( a \)


  We define the Mahloness function \( m \) as follows:

  \( (a\in T_m)\mapsto m(a)\in T_m \)

  01. If \( a=\overline{0}\vee a=K \), return \( \overline{0} \)

  02. If \( a=a_0+a_1\land a_0\in T_s\land a_1\in T_m\setminus\{\overline{0}\} \), return \( \overline{0} \)

  03. If \( a=\varphi_{a_0}(a_1)\land a_0,a_1\in T_m \), return \( \overline{0} \)

  04. If \( a=\Omega_{a_0}\land a_0\in T_m \), then:

  04.01. If \( a_0=Suc(a_1)\land a_1\in T_m\setminus\{\overline{0}\} \), return \( \overline{1} \)

  04.02. Else, return \( \overline{0} \)

  05. If \( a=\Xi(a_0)\land a_0\in T_m \), return \( a_0 \)

  06. If \( a=\Psi_{a_0}^{a_1}(a_2)\land a_0,a_1,a_2\in T_m \), return \( a_1 \)

  \( (a\in T_m,b\in T_m)\mapsto\lambda_b(a)\in T_m \)

  05.01. If \( a<b \), return \( \overline{0} \)

  06.01. If \( a<b \), return \( \overline{0} \)

  \( (a\in T_m)\mapsto (a\in T)\in\{\mathbb{T},\mathbb{F}\} \)

  02. If \( a=a_0+a_1\land a_0\in T_s\land a_1\in T_m\setminus\{\overline{0}\} \), then:

  02.02. If \( a_0\in T\land a_1\in T \), then:

  02.02.01. If \( a_1\in T_s \), return \( \neg a_0<a_1 \)

  04. If \( a=\Omega_{a_0}\land a_0\in T_m \), return \( a_0\in T\land a_0<a\land a_0<K\land a_0\neq\overline{0} \)

  We define the limit of the system \( A=\Gamma_{K+1} \). (Denoted as \( K^\Gamma \) in the paper)

  \( \Gamma_{K+1} \) is also in normal form.

  For all ordinals in this system \( \alpha\in T \), \( \alpha<\Gamma_{K+1} \)

  Ordinal \( \alpha \) is countable if and only if \( \alpha<\Omega_\overline{1} \).

  The countable limit ordinal exclusive to the command "fgh" or "sgh" is \( \Psi_{\Omega_\overline{1}}^\overline{0}(\Gamma_{K+1}) \), written in computer code as "(R(W(v,)),,A)".

Operations

  We define the normalization function \( Norm \) to normalize an ordinal to its normal form:

  \( (a\in T_m)\mapsto Norm(a)\in T \)

  01. If \( a\in T \), return \( a \)

  02. If \( \neg a\in T \), then:

  02.01. If \( a=a_0+a_1\land a_0\in T_s\land a_1\in T_m\setminus\{\overline{0}\} \), return \( Norm(a_0)+Norm(a_1) \)

  02.02. If \( a=\varphi_{a_0}(a_1)\land a_0,a_1\in T_m \), then:

  02.02.01. If \( a=a_1 \), return \( a_1 \)

  02.02.02. If \( a=a_0 \), return \( a_0 \)

  02.02.03. If \( a\neq a_0\land a\neq a_1 \), return \( a \)

  02.03. If \( a=\Omega_{a_0}\land a_0\in T_m \), return \( a_0 \)

  02.04. Else, return \( a \)


  We define addition \( Add \) as follows:

  \( (a\in T,b\in T,c\in T\cup\{0\})\mapsto Add(a,b,c)\in T \) (Note that \( 0 \) is a numerical value and it's DIFFERENT from the symbol \( \overline{0} \))

  01. If \( a=\overline{0} \), return \( b \)

  02. If \( a\neq\overline{0}\land b=\overline{0} \), return \( a \)

  03. If \( a=a_0+a_1\land b=b_0+b_1\land a_0,b_0\in T_s\land a_1,b_1\in T_m\setminus\{\overline{0}\} \), then:

  03.01. If \( c=0 \), then:

  03.01.01. If \( a_0<b_0 \), return \( b \)

  03.01.02. If \( \neg a_0<b_0 \), return \( a_0+Add(a_1,b,0) \)

  03.02. If \( c\in T \), then:

  We define the normal fixed point function \( FP \) to determine whether a term is a normal fixed point. Although there are no fixed points of the \( \Xi \) and \( \Psi \) functions, the idea is to determine whether for all \( a<b,f(a)<b \). Where \( a,b \) are arbitrary terms and \( f \) is an arbitrary function. For example, \( \Psi_\Omega^0(\Omega) \) is not a fixed point of the function \( \alpha\mapsto\Psi_\Omega^0(\alpha) \), but it's true that for all \( \alpha<\Psi_\Omega^0(\Omega),\Psi_\Omega^0(\alpha)<\Psi_\Omega^0(\Omega) \):

  \( (a\in T,b\in T)\mapsto FP_b(a)\in\{\mathbb{T},\mathbb{F}\} \)

  01. Let \( t=t_b(a) \), \( r=RP_b(a) \)

  02. If \( \neg t\in T_s \), return \( \mathbb{F} \)

  03. Else if \( \neg r\in T \), return \( \mathbb{F} \)

  04. Else if \( t=\Psi_{t_0}^{t_1}(t_2)\land t_0,t_1,t_2\in T_m \), then:

  04.01. If \( t_2<r \), return \( \mathbb{F} \)

  04.02. Else if \( t<\Psi_b^\overline{0}(r) \), return \( \mathbb{F} \)

  04.03. Else if \( t_0=_*b \), return \( \mathbb{T} \)

  04.04. Else if \( FP_b(RP_b(a,t_0)) \), return \( \mathbb{T} \)

  04.05. Else, return \( \mathbb{F} \)

  05. Else, return \( \mathbb{F} \)


  We define the Mahlo fixed point function \( \Xi FP \) to determine whether a term is a Mahlo fixed point:

  \( (a\in T)\mapsto \Xi FP(a)\in\{\mathbb{T},\mathbb{F}\} \)

  01. Let \( t=t_K(a) \), \( r=RP_K(a) \), \( R=Root(t)=\Xi(R_0) \)

  02. If \( \neg t\in T_s \), return \( \mathbb{F} \)

  03. Else if \( \neg r\in T \), return \( \mathbb{F} \)

  04. Else if \( (t=\Xi(t_0)\vee t=\Psi_{t_0}^{t_1}(t_2))\land t_0,t_1,t_2\in T_m \), then:

  04.01. If \( R_0<r \), return \( \mathbb{F} \)

  04.02. Else if \( t<\Psi_{\Xi(r)}^\overline{0}(S(r)) \), return \( \mathbb{F} \)

  04.03. Else, return \( \mathbb{T} \)

  05. Else, return \( \mathbb{F} \)


  We define the collapsing fixed point function \( \Psi FP \) to determine whether a term is a collapsing fixed point:

  \( (a\in T,b\in T,c\in T)\mapsto \Psi FP_b^c(a)\in\{\mathbb{T},\mathbb{F}\} \)

  01. Let \( t=t_K(c) \), \( r=RP_K(c) \), \( R=Root_b(t) \)

  02. If \( \neg t\in T_s \), return \( \mathbb{F} \)

  03. Else if \( \neg r\in T \), return \( \mathbb{F} \)

  04. Else if \( t=\Psi_{t_0}^{t_1}(t_2)\land t_0,t_1,t_2\in T_m \), then:

  04.01. If \( R=\overline{0} \), return \( \mathbb{F} \)

  04.02. Else if \( R=\Psi_{R_0}^{R_1}(R_2) \), then:

  04.02.01. If \( t<\Psi_b^\overline{0}(a) \), return \( \mathbb{F} \)

  04.02.02. If \( \neg t<\Psi_b^\overline{0}(a) \), then:

  04.02.02.01. If \( a<R_1 \), return \( \mathbb{T} \)

  04.02.02.02. If \( \neg a<R_1 \), then:

  04.02.02.02.01. If \( R_1<r \), return \( \mathbb{F} \)

  04.02.02.02.02. If \( \neg R_1<r\land R<\Psi_b^r(a) \), return \( \mathbb{F} \)

  04.02.02.02.03. If \( \neg R_1<r\land \neg R<\Psi_b^r(a) \), return \( \mathbb{T} \)

  04.03. Else, return \( \mathbb{F} \)

  05. Else, return \( \mathbb{F} \)

  03. Else, return \( \mathbb{T} \)

  We define the cofinality function \( Cof \) as follows:

  \( (a\in T)\mapsto Cof(a)\in T \)

  01. If \( a=\overline{0}\vee a=K \), return \( a \)

  02.02.02. Else if \( Cof(a_L)<b \), return \( Cof(a_L) \)

  02.02.03. Else, return \( \overline{\omega} \)

  02.03.01. If \( c=\overline{0}\land Cof(a_R)=\overline{1} \), then:

  02.03.01.01. If \( \overline{1}<Cof(m_R) \), then:

  02.03.01.01.01. If \( \neg\kappa FP(b)\vee\lambda(b)<a \), return \( Cof(m) \)

  02.03.01.01.02. If \( \kappa FP(b)\land\neg\lambda(b)<a \), return \( CP_{b_0}(b_2,m_R) \)

  02.03.01.02. Else if \( \lambda(b)<a \), return \( CP_{b_0}(b_2,\Psi_b^{m[\overline{0}]}(a[\overline{0}])) \)

  02.04. Else if \( c\neq\overline{0} \), return \( \overline{\omega} \)

  02.05. Else if \( \neg\overline{1}<Cof(m_R) \), return \( \overline{\omega} \)

  02.06. Else if \( Cof(a_R)\neq\overline{1} \), return \( \overline{\omega} \)

  02.07. Else if \( \kappa FP(b)\land\neg\lambda(b)<a \), return \( \overline{\omega} \)

  02.08. Else, return \( Cof(m) \)

  03. If \( m_L\neq\overline{0} \), then:

  03.01. If \( a_R=\overline{0} \), then:

  03.02.02.01. If \( Cof(m_L)<K \), return \( Cof(m_L) \)

  03.02.02.02. If \( \neg Cof(m_L)<K \), return \( \overline{\omega} \)

  03.03. If \( \neg Cof(a_R)<\overline{\omega} \), return \( Cof(a) \)
