Definition

  Buchholz's \( \psi \) refers to an ordinal collapsing function proposed by Wilfried Buchholz in 1986[1]. The definition of the function \( \psi_\alpha(\beta) \) is as follows:

  \( \begin{align} C_0(\alpha,\beta) := & \begin{cases} 1 & \alpha=0 \\ \Omega_\alpha & \alpha>0 \end{cases} \\ C_{n+1}(\alpha,\beta) := & \{\gamma|\gamma=\gamma_0+\gamma_1\land\gamma_0,\gamma_1\in C_n(\alpha,\beta)\} \\ \cup & \{\gamma|\gamma=\psi_{\alpha'}(\beta')\land\alpha'<\omega+1\land\beta'\in C_n(\alpha,\beta)\land\beta'<\beta\} \\ C(\alpha,\beta) := & \bigcup_{n<\omega}{C_n(\alpha,\beta)}=C_0(\alpha,\beta)\cup C_1(\alpha,\beta)\cup C_2(\alpha,\beta)\cup\cdots\cup C_m(\alpha,\beta)\cup\cdots \\ \psi_\alpha(\beta) := & min(Ord\setminus C(\alpha,\beta)) \end{align} \)


  Buchholz's \( \psi \) can be considered one of the easiest OCF. The reason is as follows:

  1. Buchholz's \( \psi \) is monotonically increasing for both variables. Meaning that the return value of the function is easier to predict.

  2. Unlike most of other OCFs, Buchholz's \( \psi \) has the property: \( \psi_0(0)=1 \) and \( \psi_\alpha(\beta+1)=\psi_\alpha(\beta)*\omega \) if \( \psi_\alpha(\beta+1)>\psi_\alpha(\beta) \), meaning that every ordinal that Buchholz's \( \psi \) can represent can be constructed only using zero, addition and the function.

  The fundamental sequence system used in this website is defined as follows, note that every term has to be in normal form. Due to the complexity, the normal form will be defined in the "Ordinal Notation" section:

  01. If \( \alpha=0 \), then \( \alpha[n]:=0 \) and \( Cof(\alpha):=0 \)

  02. If \( \alpha=\psi_{\beta_1}(\gamma_1)+\psi_{\beta_2}(\gamma_2)+\cdots+\psi_{\beta_m}(\gamma_m) \) and \( m>1 \), then:

  02.01. \( \alpha[n]:=\psi_{\beta_1}(\gamma_1)+\psi_{\beta_2}(\gamma_2)+\cdots+\psi_{\beta_m}(\gamma_m)[n] \) and \( Cof(\alpha):=Cof(\psi_{\beta_m}(\gamma_m)) \)

  03. If \( \alpha=\psi_0(0)=1 \), then \( \alpha[n]:=0 \) and \( Cof(\alpha):=1 \)

  04. If \( \alpha=\psi_{\beta}(0)=\Omega_\beta \) and \( Cof(\beta)=1 \), then \( \alpha[n]:=n \) and \( Cof(\alpha):=\alpha \)

  05. If \( \alpha=\psi_\omega(0)=\Omega_\omega \), then \( \alpha[n]:=\psi_n(0) \) and \( Cof(\alpha):=\psi_0(\psi_0(0))=\omega \)

  06. If \( \alpha=\psi_{\beta}(\gamma) \) and \( Cof(\gamma)=1 \), then \( \alpha[n]:=\psi_{\beta}(\gamma[0])*n \) and \( Cof(\alpha):=\omega \)

  07. If \( \alpha=\psi_{\beta}(\gamma) \) and \( 1<Cof(\gamma)\le\Omega_\beta \), then \( \alpha[n]:=\psi_{\beta}(\gamma[n]) \) and \( Cof(\alpha):=Cof(\gamma) \)

  08. If \( \alpha=\psi_{\beta}(\gamma) \) and \( \Omega_\beta<Cof(\gamma) \), then \( \alpha[0]:=0 \) and \( \alpha[n+1]:=\psi_{\beta}(Re(\gamma,n)) \) and \( Cof(\alpha):=\omega \),

  08.01. where \( Re(\gamma,0):=\gamma[0] \) and \( Re(\gamma,n+1):=\gamma[\psi_{\delta}(Re(\gamma,n))] \) and \( \Omega_{\delta+1}=Cof(\gamma) \)

  09. If \( \alpha=\varepsilon_{\Omega_\omega+1} \), then \( \alpha[0]:=0 \) and \( \alpha[n+1]:=\psi_\omega(\alpha[n]) \) and \( Cof(\alpha):=\omega \)


  Note that Wilfried Buchholz also defined his fundamental sequence system in his paper, which is slightly different from the definition above.

  The countable limit of this OCF, denoted as \( \psi_0(\varepsilon_{\Omega_\omega+1}) \), is called the Takeuti-Feferman-Buchholz ordinal (TFBO). It's the PTO of \( \Pi_1^1-CA+BI \).

Ordinal Notation

Notation

  We define \( S \) to be the set that contains all the strings composed of the following symbols:

  Constant "\( \overline{0} \)",

  Function symbol "\( \psi \)",

  And other symbols "\( + \)", "\( ( \)", "\( ) \)" and "\( , \)".


  We define the subsets of \( S \), \( T_m \) and \( T_s \) as follows:

  01. \( \overline{0}\in T_m \)

  02. For all \( a\in T_s \) and \( b\in T_m\setminus\{\overline{0}\} \), \( a+b\in T_m \)

  03. For all \( a,b\in T_m \), \( \psi(a,b)\in T_m \) and \( \psi(a,b)\in T_s \)


  Throughout this article, we use the following abbreviation:

  01. \( \psi_a(b):=\psi(a,b) \)

  02. \( \overline{1}:=\psi_\overline{0}(\overline{0}) \)

  03. \( \overline{\omega}:=\psi_\overline{0}(\overline{1}) \)

Ordering

  We define the binary relation \( < \) as follows:

  \( (a\in T_m,b\in T_m)\mapsto a<b\in\{\mathbb{T},\mathbb{F}\} \)

  01. If \( b=\overline{0} \), return \( \mathbb{F} \)

  02. If \( a=\overline{0}\land b\neq\overline{0} \), return \( \mathbb{T} \)

  03. If \( a=a_0+a_1\land b=b_0+b_1\land a_0,b_0\in T_s\land a_1,b_1\in T_m\setminus\{\overline{0}\} \), then:

  03.01. If \( a_0<b_0 \), return \( \mathbb{T} \)

  03.02. If \( \neg a_0<b_0 \), return \( \neg b_0<a_0\land a_1<b_1 \)

  04. If \( a=a_0+a_1\land b,a_0\in T_s\land a_1\in T_m\setminus\{\overline{0}\} \), return \( a_0<b \)

  05. If \( b=b_0+b_1\land a,b_0\in T_s\land b_1\in T_m\setminus\{\overline{0}\} \), return \( \neg b_0<a \)

  06. If \( a=\psi_{a_0}(a_1)\land b=\psi_{b_0}(b_1)\land a_0,a_1,b_0,b_1\in T_m \), return \( a_0<b_0\vee(\neg b_0<a_0\land a_1<b_1) \)


  We define all the other comparison symbols:

  \( (a\in T_m,b\in T_m)\mapsto a>b\in\{\mathbb{T},\mathbb{F}\} \)

  Return \( b<a \)


  \( (a\in T_m,b\in T_m)\mapsto a\le b\in\{\mathbb{T},\mathbb{F}\} \)

  Return \( \neg b<a \)


  \( (a\in T_m,b\in T_m)\mapsto a\ge b\in\{\mathbb{T},\mathbb{F}\} \)

  Return \( \neg a<b \)


  \( (a\in T_m,b\in T_m)\mapsto a=_*b\in\{\mathbb{T},\mathbb{F}\} \)

  Return \( \neg a<b\land\neg b<a \)


  \( (a\in T_m,b\in T_m)\mapsto a\neq_*b\in\{\mathbb{T},\mathbb{F}\} \)

  Return \( a<b\vee b<a \)


  Note that \( = \) and \( \neq \) represents symbolic equivalence while \( =_* \) and \( \neq_* \) represents numerical equivalence.

Normal Form

  We define the function \( \in C \) to determine whether an ordinal \( c \) is an element of \( C(a,b) \):

  \( (c\in T_m,a\in T_m,b\in T_m)\mapsto c\in C(a,b)\in\{\mathbb{T},\mathbb{F}\} \)

  01. If \( c<\psi_a(\overline{0}) \), return \( \mathbb{T} \)

  02. If \( \neg c<\psi_a(\overline{0}) \), then:

  02.01. If \( c=c_0+c_1\land c_0\in T_s\land c_1\in T_m\setminus\{\overline{0}\} \), return \( c_0\in C(a,b)\land c_1\in C(a,b) \)

  02.02. If \( c=\psi_{c_0}(c_1)\land c_0,c_1\in T_m \), return \( c_0<\overline{\omega}+\overline{1}\land c_1\in C(a,b)\land c_1<b \)


  We define the function \( \in T \) to determine whether an ordinal is in normal form:

  \( (a\in T_m)\mapsto (a\in T)\in\{\mathbb{T},\mathbb{F}\} \)

  01. If \( a=\overline{0} \), return \( \mathbb{T} \)

  02. If \( a=a_0+a_1\land a_0\in T_s\land a_1\in T_m\setminus\{\overline{0}\} \), then:

  02.01. If \( \neg a_0\in T\vee\neg a_1\in T \), return \( \mathbb{F} \)

  02.02. If \( a_0\in T\land a_1\in T \), then:

  02.02.01. If \( a_1\in T_s \), return \( \neg a_0<a_1 \)

  02.02.02. If \( a_1=a_2+a_3\land a_2\in T_s\land a_3\in T_m\setminus\{\overline{0}\} \), return \( \neg a_0<a_2 \)

  03. If \( a=\psi_{a_0}(a_1)\land a_0,a_1\in T_m \), return \( a_0\in T\land a_0<\overline{\omega}+\overline{1}\land a_1\in T\land a_1\in C(a_0,a_1) \)


  We define the limit of the system \( A=\varepsilon_{\Omega_\omega+1} \).

  \( \varepsilon_{\Omega_\omega+1} \) is also in normal form.

  For all ordinals in this system \( \alpha\in T \), \( \alpha<\varepsilon_{\Omega_\omega+1} \)

  Ordinal \( \alpha \) is countable if and only if \( \alpha<\psi_\overline{1}(\overline{0}) \).

  The countable limit ordinal exclusive to the command "fgh" or "sgh" is \( \psi_\overline{0}(\varepsilon_{\Omega_\omega+1}) \), written in computer code as "(,A)".

Operations

  We define addition \( Add \) as follows:

  \( (a\in T,b\in T)\mapsto Add(a,b)\in T \)

  01. If \( a=\overline{0} \), return \( b \)

  02. If \( a\neq\overline{0}\land b=\overline{0} \), return \( a \)

  03. If \( a\neq\overline{0}\land b\neq\overline{0} \), return \( a+b \)

Cofinality

  We define the cofinality function \( Cof \) as follows:

  \( (a\in T)\mapsto Cof(a)\in T \)

  01. If \( a=\overline{0} \), return \( a \)

  02. If \( a=a_0+a_1\land a_0\in T_s\land a_1\in T_m\setminus\{\overline{0}\} \), return \( Cof(a_1) \)

  03. If \( a=\psi_{a_0}(a_1)\land a_0,a_1\in T_m \), then:

  03.01. If \( a_1=\overline{0} \), then:

  03.01.01. If \( a_0\neq\overline{\omega} \), return \( a \)

  03.01.02. If \( a_0=\overline{\omega} \), return \( \overline{\omega} \)

  03.02. If \( Cof(a_1)=\overline{1} \), return \( \overline{\omega} \)

  03.03. If \( \neg Cof(a_1)<\overline{\omega} \), then:

  03.03.01. If \( \neg\psi_{a_0}(0)<Cof(a_1) \), return \( Cof(a_1) \)

  03.03.02. If \( \psi_{a_0}(0)<Cof(a_1) \), return \( \overline{\omega} \)


  We also define that \( Cof(\varepsilon_{\Omega_\omega+1})=\overline{\omega} \).

Fundamental Sequence

  We define the fundamental sequence \( [] \) as follows:

  \( (a\in T,n\in T)\mapsto a[n]\in T \)

  01. If \( a=\overline{0} \), return \( \overline{0} \)

  02. If \( a=a_0+a_1\land a_0\in T_s\land a_1\in T_m\setminus\{\overline{0}\} \), return \( Add(a_0,a_1[n]) \)

  03. If \( a=\psi_{a_0}(a_1)\land a_0,a_1\in T_m \), then:

  03.01. If \( a_1=\overline{0} \), then:

  03.01.01. If \( a_0=\overline{0} \), return \( \overline{0} \)

  03.01.02. If \( Cof(a_0)=\overline{1} \), return \( n \)

  03.01.03. If \( a_0=\overline{\omega} \), return \( \psi_n(\overline{0}) \)

  03.02. If \( Cof(a_1)=\overline{1} \), then:

  03.02.01. If \( n=\overline{0} \), return \( \overline{0} \)

  03.02.02. If \( n\neq\overline{0} \), return \( Add(a[n[\overline{0}]], \psi_{a_0}(a_1[\overline{0}])) \)

  03.03. If \( \neg Cof(a_1)<\overline{\omega} \), then:

  03.03.01. If \( \neg\psi_{a_0}(0)<Cof(a_1) \), return \( \psi_{a_0}(a_1[n]) \)

  03.03.02. If \( \psi_{a_0}(0)<Cof(a_1) \), then:

  03.03.02.01. If \( n=\overline{0} \), return \( \overline{0} \)

  03.03.02.02. If \( n\neq\overline{0} \), return \( \psi_{a_0}(Re(a_1,n[\overline{0}])) \)


  We define the recursion function \( Re \) as follows:

  \( (a\in T,n\in T)\mapsto Re(a,n)\in T \)

  01. If \( n=\overline{0} \), return \( a[\overline{0}] \)

  02. If \( n\neq\overline{0}\land Cof(a)=\psi_{a_0}(0)\land a_1=a_0[\overline{0}] \), return \( a[\psi_{a_1}(Re(a,n[\overline{0}]))] \)


  We define the strong fundamental sequence \( {[]}^* \) in case we want an ordinal of uncountable cofinality to expand as a smaller ordinal with countable cofinality:

  \( (a\in T,n\in T)\mapsto a{[n]}^*\in T \)

  01. If \( \neg\overline{\omega}<Cof(a) \), return \( a[n] \)

  02. If \( \overline{\omega}<Cof(a)\land Cof(a)=\psi_{a_0}(0)\land a_1=a_0[\overline{0}] \), return \( a[\psi_{a_1}(\varepsilon_{\Omega_\omega+1}[n])] \)


  We also define the fundamental sequence of \( \varepsilon_{\Omega_\omega+1} \) as follows:

  \( (n\in T)\mapsto\varepsilon_{\Omega_\omega+1}[n]\in T \)

  01. If \( n=\overline{0} \), return \( \overline{0} \)

  02. If \( n\neq\overline{0} \), return \( \psi_\overline{\omega}(\varepsilon_{\Omega_\omega+1}[n[\overline{0}]]) \)
